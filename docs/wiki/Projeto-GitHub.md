# Projeto GitHub (board)

Tudo que é trabalho do **Dino** pode ser acompanhado na interface do GitHub.

## Project (v2)

- **Nome:** Dino — produto & engenharia  
- **URL:** [github.com/users/adventurelabsbrasil/projects/7](https://github.com/users/adventurelabsbrasil/projects/7)

O repositório **`adventurelabsbrasil/dino`** está **vinculado** ao projeto. Issues e PRs podem ser adicionados ao board:

- Na issue/PR: painel direito → **Projects** → selecione o projeto.
- Ou arraste cards no board e use colunas **Status** (Backlog / Ready / In progress / Done — ajuste às suas necessidades).

### Boas práticas

1. Toda nova feature ou bug: **issue** com label (`plano-dino`, `enhancement`, `bug`, etc.).
2. PRs: vincular à issue (`Closes #N` na descrição).
3. Board: mover card conforme estado; filtrar por **label** ou **milestone** se criar.
